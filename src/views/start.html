GitHub Requests

The GitHub Requests application is a tool designed to streamline GitHub organisation management, which involves automatically managing requests to access organisation resources. Further documentation can be found here.

Before you start:

  • New users must have a GitHub handle.

  • New users must be associated with an existing team (with the exception of outside collaborators), otherwise, the new team must be created in the same request.

  • New teams must have at least one maintainer.

  • New users that are not permanent staff must provide an expected end date of their contract.
